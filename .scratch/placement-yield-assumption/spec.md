# Placement Yield Assumption

Status: ready-for-agent

The planned Web convertible-placement strategy will use a global, locally persisted expected-listing-premium assumption. It will default to 30%, accept 30% through 100% in 10% increments, and drive the displayed expected profit, safety pad, score, rating, and composite ranking. The API source fields will remain available but will not control those derived values.

See `../../openspec/changes/placement-yield-assumption/` for the authoritative proposal, requirements, design, and task list.
