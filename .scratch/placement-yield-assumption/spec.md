# Placement Yield Assumption

Status: ready-for-agent

The Web convertible-placement strategy uses a global, locally persisted expected-listing-premium assumption. It defaults to 30%, accepts 30% through 100% in 10% increments, and drives the displayed expected profit, safety pad, score, rating, and composite ranking. The API source fields remain available but do not control those derived values.

See `../../openspec/changes/placement-yield-assumption/` for the authoritative proposal, requirements, design, and task list.
