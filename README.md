# Trading Toolkit Web

🧠 **A/H 股套利策略数据支撑工作台** —— 一个基于认知科学设计的投资工具，帮助你在使用中学习，在学习中提升，形成良性的认知闭环。

## 🌟 核心设计理念

### Learning Loop（学习循环）

### 认知科学理论支撑

| 理论 | 应用场景 | 设计效果 |
|------|---------|---------|
| **认知负荷理论** | 参数分组展示，每组不超过 3-4 个 | 避免信息过载 |
| **生成效应** | 参数敏感度滑块，主动探索 | 动手计算加深记忆 |
| **渐进式披露** | 信息分层，按需展开 | 从结论到细节，循序渐进 |
| **间隔重复** | 公式卡片定期提醒 | 强化长期记忆 |
| **实例学习** | 历史案例 + 分步计算 | 降低学习难度 |

### 策略难度分级

| 级别 | 标签 | 门槛 | 适用人群 |
|------|------|------|---------|
| 🟢 入门 | 低风险、低门槛 | ≤ 1万元 | 新手投资者 |
| 🟡 进阶 | 中等风险、有门槛 | 1万 - 50万 | 有经验投资者 |
| 🔴 专业 | 高风险、高门槛 | ≥ 50万 | 机构/专业投资者 |

## 📈 策略覆盖

### 🟢 入门级策略（无门槛）
- **LOF 基金溢价套利**：利用场内场外价差获利
- **可转债打新/配售**：低风险打新，安全垫计算
- **港股打新**：港股 IPO 申购，PE 估值分析
- **封闭式基金折价套利**：折价回归，确定性收益

### 🟡 进阶级策略（有一定门槛）
- **A/H 股溢价套利**：跨市场价差，港股通门槛 50万
- **ETF 二级市场折价买入**：折价回归策略
- **可转债折价转股套利**：转股期内折价套利

### 🔴 专业级策略（高门槛）
- **ETF 一级市场申赎套利**：百万级资金门槛
- **股指期货基差套利**：50万+门槛，专业知识

## 🛠️ 工作台辅助功能

### 🧮 悬浮计算器
实时计算策略收益，支持自定义参数，显示计算公式

### 📝 悬浮笔记
记录观察、想法、操作计划，自动关联策略和时间

### 📊 悬浮公式库
按策略分类的公式手册，包含公式、示例、详细说明

### ⏱️ 时间日期标注
所有数据带时间戳，支持历史趋势对比，确保信息时效性

## 🚀 快速开始

### 开发环境
```bash
# 安装依赖（使用国内镜像加速）
npm install --registry=https://registry.npmmirror.com

# 启动开发服务器
npm run dev
```

### 构建部署
```bash
# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
