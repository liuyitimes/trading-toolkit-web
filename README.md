### 认知科学 & 学习理论支撑

我们的设计融合了 **认知科学、高效学习法和交互设计** 三大领域的经典理论，将学习原理编织进每一次交互中。

#### 🧠 认知科学理论

| 理论 | 核心原理 | 设计应用 |
|------|---------|---------|
| **认知负荷理论** (Sweller, 1988) | 工作记忆容量有限（4±1 组块），超载则学习效率骤降 | 参数分组展示，每组 ≤ 4 个核心指标 |
| **组块理论 / 米勒定律** (Miller, 1956) | 人类短期记忆只能处理 7±2 个信息单元 | 信息分层，每层聚焦 3-5 个关键数据 |
| **双编码理论** (Paivio, 1971) | 视觉 + 文字双重编码，记忆效果优于单一编码 | 公式配图表，数据配颜色标签 |
| **生成效应** (Slamecka & Graf, 1978) | 主动生成信息比被动接收记忆更深 | 参数滑块让用户动手计算，非被动观看 |
| **检索练习** (Roediger & Karpicke, 2006) | 主动回忆比重复阅读记忆保留率高 50% | 公式卡片填空，让用户回忆而非重读 |
| **即时反馈** (Hattie & Timperley, 2007) | 实时反馈是影响学习效果的最强因素之一 | 滑块调整参数即刻显示收益变化 |

#### 📚 高效学习法

| 方法 | 核心原理 | 设计应用 |
|------|---------|---------|
| **学习金字塔** (National Training Labs) | 被动听讲仅保留 5%，动手实践保留 75% | 工作台交互优先，减少纯阅读 |
| **费曼技巧** (Feynman) | 能用简单语言解释清楚，才算真正理解 | 策略原理用通俗语言说明，公式拆解分步演示 |
| **刻意练习** (Ericsson, 1993) | 在"学习区"内反复练习，配以及时反馈 | 敏感度分析提供精确反馈，策略难度分级引导 |
| **最近发展区 / 脚手架** (Vygotsky / Bruner) | 在帮助下完成略高于当前能力的任务 | 新手引导 → 公式提示 → 自主操作，逐步撤除辅助 |
| **间隔重复** (Ebbinghaus, 1885) | 在遗忘临界点复习，记忆效果最佳 | 公式卡片按科学周期提醒复习 |

#### 🎨 交互设计原则

| 原则 | 核心原理 | 设计应用 |
|------|---------|---------|
| **渐进式披露** (Nielsen, 2006) | 信息按需展示，用多少展示多少 | 核心结论 → 展开公式 → 深度解析，三级信息架构 |
| **自我决定理论** (Ryan & Deci, 2000) | 自主性、胜任感、归属感驱动内在动机 | 自由探索参数、难度分级带来成就感 |
| **希克定律** (Hick, 1952) | 选择越多，决策越慢 | 策略按难度分组，每页只呈现核心操作 |
| **菲茨定律** (Fitts, 1954) | 目标越大越近，操作越快 | 悬浮工具栏固定位置，一键触达核心功能 |

#### 📊 理论映射关系

```
认知负荷理论 → 参数分组 ≤ 4 个
组块理论     → 信息分层聚焦
生成效应     → 参数滑块主动探索
检索练习     → 公式填空回忆
学习金字塔   → 动手实践优先
费曼技巧     → 通俗化策略说明
刻意练习     → 敏感度分析 + 即时反馈
最近发展区   → 脚手架式引导
间隔重复     → 科学复习周期
渐进式披露   → 三级信息架构
自我决定理论 → 自主探索 + 难度分级
即时反馈     → 实时计算展示
```

### 策略难度分级

| 级别 | 标签 | 门槛 | 适用人群 |
|------|------|------|---------|
| 🟢 入门 | 低风险、低门槛 | ≤ 1万元 | 新手投资者 |
| 🟡 进阶 | 中等风险、有门槛 | 1万 - 50万 | 有经验投资者 |
| 🔴 专业 | 高风险、高门槛 | ≥ 50万 | 机构/专业投资者 |

## 📈 策略覆盖

### 🟢 入门级策略（无门槛）
- **LOF 基金溢价套利**：利用场内场外价差获利
- **可转债打新/配售**：低风险打新，安全垫计算
- **港股打新**：港股 IPO 申购，PE 估值分析
- **封闭式基金折价套利**：折价回归，确定性收益

### 🟡 进阶级策略（有一定门槛）
- **A/H 股溢价套利**：跨市场价差，港股通门槛 50万
- **ETF 二级市场折价买入**：折价回归策略
- **可转债折价转股套利**：转股期内折价套利

### 🔴 专业级策略（高门槛）
- **ETF 一级市场申赎套利**：百万级资金门槛
- **股指期货基差套利**：50万+门槛，专业知识

## 🛠️ 工作台辅助功能

### 🧮 悬浮计算器
实时计算策略收益，支持自定义参数，显示计算公式

### 📝 悬浮笔记
记录观察、想法、操作计划，自动关联策略和时间

### 📊 悬浮公式库
按策略分类的公式手册，包含公式、示例、详细说明

### ⏱️ 时间日期标注
所有数据带时间戳，支持历史趋势对比，确保信息时效性
